# Currency Converter

## Description
The Currency Converter is a Java console-based application that enables users to convert an amount from one currency to another using predefined exchange rates. The application supports multiple international currencies and provides a simple, menu-driven interface for quick and accurate currency conversion.

## Features
- Convert between multiple currencies (INR, USD, EUR, GBP, JPY, etc.)
- User-friendly console interface
- Supports repeated conversions
- Displays converted amount instantly
- Input validation for currency selection and amount
- Easy-to-understand menu-driven program

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Conditional Statements
- Loops
- Methods

## How to Run
1. Clone this repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile the `CurrencyConverter.java` file.
4. Run the program.
5. Select the source and target currencies.
6. Enter the amount to convert.
7. View the converted amount.

## Supported Currencies
- INR (Indian Rupee)
- USD (US Dollar)
- EUR (Euro)
- GBP (British Pound)
- JPY (Japanese Yen)
